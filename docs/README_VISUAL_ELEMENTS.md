## GitHub Markdown Limitations

**What Works:**
✅ Badges (shields.io)
✅ Tables
✅ Code blocks with syntax highlighting
✅ Emoji
✅ HTML for alignment (div, img)
✅ Anchor links

**What Doesn't Work:**
❌ CSS styling
❌ JavaScript interactions
❌ Custom fonts
❌ Video embeds (use GIFs instead)
❌ iframes
